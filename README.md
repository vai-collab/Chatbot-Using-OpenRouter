# 🤖 ChatGPT Chatbot using OpenRouter (Streamlit)

## 📌 Project Description

This project is an **AI Chatbot Web Application** built using **Python and Streamlit** that integrates with the **OpenRouter API** to generate intelligent responses similar to ChatGPT.

The application provides a **simple and interactive chat interface** where users can ask questions and receive AI-generated responses in real time.

The chatbot can help users with **programming questions, topic-wise notes, explanations, coding help, and general queries**.

---


## 🚀 Features

* 💬 Interactive chatbot interface
* 🤖 AI-powered responses using OpenRouter API
* ⚡ Real-time response generation
* 🧠 Topic-wise notes and explanations
* 📜 Scrollable chat conversation
* 🎨 Clean and simple Streamlit UI

## ⚙️ Installation

### 1️⃣ Clone the Repository

```
git clone https://github.com/yourusername/chatgpt-openrouter-chatbot.git
```

### 2️⃣ Go to the Project Folder

```
cd chatgpt-openrouter-chatbot
```

### 3️⃣ Install Required Libraries

```
pip install -r requirements.txt
```

---

## 🔑 Setup OpenRouter API Key

1. Visit: https://openrouter.ai
2. Create an account
3. Generate your **API Key**

Add the API key in your code:

```
OPENROUTER_API_KEY = "your_api_key_here"
```

---

## ▶️ Run the Application

Run the Streamlit application using:

```
streamlit run app.py
```

Open in browser:

```
http://localhost:8501
```
